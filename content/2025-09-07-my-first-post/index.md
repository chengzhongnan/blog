## 为什么选择静态网站

静态网站生成器有许多优点：

- **速度快**：没有数据库查询，没有服务器端渲染，只有纯粹的 HTML 文件。
- **安全性高**：攻击面大大减少。
- **部署简单**：可以将文件托管在任何地方，如 Cloudflare Pages 或 GitHub Pages。
